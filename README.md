# HAZOP Copilot MVP

- **Project workspace for HAZOP studies:** Manage projects, team members, tasks, files, and HAZOP rows in one place.
- **Automatic P&ID object detection:** Detects relevant equipment and symbols from uploaded process diagrams.
- **Node detection on process diagrams:** Identifies likely HAZOP nodes/sections from P&ID context and user input to support study setup.
- **AI-assisted HAZOP analysis:** Uses detected objects, nodes, background knowledge, and past meetings to generate and improve HAZOP suggestions and analysis inputs.
- **Meeting support with AI transcript + summary:** Captures meeting content, generates transcripts and summaries, and feeds this context back to help the agent decide better nodes and stronger HAZOP analysis recommendations.
- **Download autogenerated CSV**

## Core platform and infrastructure

- **AWS S3** for secure document/file storage.
- **AWS Cognito** for authentication and user/session management.
- **AWS RDS (PostgreSQL)** for persistent project, meeting, and HAZOP data.
- **AWS Amplify** for cloud integration and application environment support.
